# Robust Spam Message Classifier

## Overview
This project builds a **robust machine learning model** to classify messages as *spam* or *ham (not spam)*.  
It goes beyond basic classification by testing the model against **noisy and adversarial inputs**, simulating real-world conditions where users may intentionally or unintentionally distort text.

---

## Objectives
- Build an accurate spam detection model
- Handle real-world noisy text (typos, variations)
- Improve robustness against adversarial inputs
- Evaluate model performance using multiple metrics

---

## Tech Stack
- **Language:** Python  
- **Libraries:** Scikit-learn, Pandas, NumPy  
- **Techniques:** NLP, TF-IDF Vectorization, Logistic Regression  

---

## Features
- ✅ Text preprocessing (lowercasing, stopword removal, cleaning)
- ✅ TF-IDF feature extraction
- ✅ Logistic Regression classifier
- ✅ Model evaluation (Accuracy, Precision, Recall, F1-score)
- ✅ Adversarial noise injection (typos, word perturbations)
- ✅ Robustness testing under noisy conditions

---

## Model Performance
- Accuracy: **98%** on balanced dataset  
- Strong precision and recall balance  
- Improved resilience to noisy inputs after tuning  

---

## 🔬 Adversarial Testing
To simulate real-world scenarios, the model was tested using:
- Random typos  
- Character swaps  
- Word perturbations  

This helped evaluate how well the model performs when inputs are not clean.
